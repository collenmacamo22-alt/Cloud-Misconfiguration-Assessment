# Remediation – Disabled Multi-Factor Authentication

## Objective

Strengthen cloud account security by requiring an additional authentication factor beyond passwords.

---

## Immediate Actions

- Enable MFA for all administrator accounts.
- Enroll all users in MFA.
- Remove inactive accounts.
- Review authentication policies.

---

## Authentication Security

- Enforce MFA through organizational policy.
- Prefer phishing-resistant authentication methods where available.
- Require MFA for remote access.
- Regularly review authentication logs.

---

## Monitoring

- Alert on repeated failed login attempts.
- Monitor changes to MFA settings.
- Investigate accounts with MFA disabled.

---

## Long-Term Recommendations

- Make MFA mandatory for all cloud services.
- Conduct periodic compliance reviews.
- Educate users on phishing and credential theft.
- Review authentication policies annually.
